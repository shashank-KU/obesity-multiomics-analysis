# Obesity Multi-omics Analysis

This repository contains the complete R analysis workflow for the manuscript:

> Longitudinal lipidomics, metabolomics and PFAS analyses from a randomized obesity intervention study.

## Contents

- Scripts_manuscript.Rmd – Complete reproducible analysis pipeline
- Scripts_manuscript.html – Rendered analysis report

## Methods

The workflow includes:

- Data preprocessing
- Quality control
- Linear mixed-effects models
- Estimated marginal means
- WGCNA
- Lipidomics
- Metabolomics
- PFAS analysis
- Multiple-testing correction
- Publication-ready figures
- Supplementary tables

## Requirements

R >= 4.3

Main packages include

- tidyverse
- lme4
- emmeans
- WGCNA
- pheatmap
- openxlsx
- ggplot2
- broom.mixed
